This repository presents an innovative project developed primarily with TypeScript.

## Installation

```bash
git clone https://github.com/Dakshx07/Daksh-Portfolio-Project.git
cd Daksh-Portfolio-Project
npm install
```

## Usage

```bash
# Start the development server
npm run dev

# Build the project for production
npm run build
```
